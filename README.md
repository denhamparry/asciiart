# Asciiart

> Demo for how to use Chainguard Images

## Demo

Take a `.png` image and convert it to asciiart exporting it into a `.txt` file.

## Setup

```sh
docker build -t ascii-art-generator .  
```

## Usage

```sh
docker run --rm -v $(pwd)/image/:/app/image/ ascii-art-generator
```
